Alin Customs Brokerage

www.alincustomsbrokerage.com

This project is a website for Alin Customs Brokerage, a duly accredited authorized Customs Brokerage firm with the Bureau of Customs based in Cebu City, Philippines.

The site was developed from ground up using Bootstrap (for mobile first web design), ReactJS (front-end JavaScript framework), NodeJS (backend scripting) and native JavaScript (for implementing the core functions). The site was also deployed using Digital Ocean Cloud with its corresponding DNS.

Roles: UI/UX Designer, Full Stack Web Developer

Technologies: HTML, CSS, JavaScript, jQuery, ReactJS, NodeJS, Digital Ocean Cloud

SUPER LECHON

www.superlechon.com

This project is a website for Super Lechon, a online delivery shop for customers (within Philippines and international) who wanted to enjoy Cebu Lechon. The site serves as an online platform for client inquiries and for gathering customer orders.

The site was developed from ground up using Bootstrap (for mobile first web design), ReactJS (front-end JavaScript framework), NodeJS (backend scripting) and native JavaScript (for implementing the core functions). The site was also deployed using Amazon Web Services EC2 with its corresponding DNS.

Roles: UI/UX Designer, Full Stack Web Developer

Technologies: HTML, CSS, JavaScript, jQuery, ReactJS, NodeJS, Amazon Web Services EC2
